## 一、设计语言

**"严肃的科技感"** —— 给律师用的工具，不要赛博朋克游戏化，但要现代、克制、有质感。

三个关键词：
1. **暗色为主**：深蓝灰底 + 微弱渐变，长时间看屏幕不疲劳。
2. **玻璃拟态克制使用**：仅顶栏、模态、关键卡片用毛玻璃，列表/表格不用。
3. **微交互而非动画堆砌**：聚焦、悬停、状态切换有动效，但不超过 250ms。

参考产品（用于校准方向，不抄）：
- **Linear**：信息密度 + 暗色 + 干净
- **Vercel Dashboard**：留白 + 渐变线条
- **Arc Browser 设置页**：玻璃质感
- **Anthropic Console**：克制的科技感

**反例**（不要走的方向）：
- ❌ 大量荧光色、霓虹灯效果（赛博朋克）
- ❌ AntD/Element 的浅色卡片堆叠（普通后台）
- ❌ 大圆角 + 厚阴影 + 渐变按钮（Web3 风格）

## 二、色板

定义在 `tailwind.config.ts` 和 `src/styles/globals.css` 的 CSS 变量中，配合 shadcn/ui 的主题系统。

### 2.1 暗色主题（默认）

```css
:root {
  /* Background */
  --bg-base:        #0A0E1A;   /* 最底层 */
  --bg-elevated:    #111727;   /* 卡片/面板 */
  --bg-overlay:     #1A2238;   /* 悬浮元素 */
  --bg-glass:       rgba(20, 28, 48, 0.6); /* 毛玻璃 */

  /* Borders */
  --border-subtle:  #1F2A44;
  --border-default: #2A3656;
  --border-strong:  #3D4D75;

  /* Text */
  --text-primary:   #E8ECF5;
  --text-secondary: #9BA8C7;
  --text-muted:     #5B6889;
  --text-disabled:  #3D4666;

  /* Brand & Accent */
  --accent-primary: #5B8DEF;   /* 法理蓝，主操作色 */
  --accent-hover:   #7BA5F5;
  --accent-glow:    rgba(91, 141, 239, 0.25);
  --accent-cyan:    #4FD1C5;   /* 辅助强调（进行中/成功） */
  --accent-violet:  #9B7BF7;   /* 仅用于 AI 相关功能 */

  /* Status */
  --status-success: #4ADE80;
  --status-warning: #FBBF24;
  --status-danger:  #F87171;
  --status-info:    #60A5FA;

  /* Severity (冲突/期限) */
  --severity-low:      #4ADE80;
  --severity-medium:   #FBBF24;
  --severity-high:     #FB923C;
  --severity-blocking: #F87171;
}
```

### 2.2 浅色主题（V1.5 提供，V1 先不做）

预留 `[data-theme="light"]` 切换，但 V1 默认强制暗色（保留 toggle 但置灰）。

### 2.3 主题色自定义（V1 开放）

`--accent-primary`（法理蓝 `#5B8DEF`）为系统默认主色。管理员可在 `/settings/appearance` 选择：

- 法理蓝（默认）`#5B8DEF`
- 法庭金 `#D4A453`（适合传统律所）
- 翡翠绿 `#3FB28F`
- 朱砂红 `#E8694D`（仅用于刑辩所，慎选）

或自定义色值（HEX）。系统会自动派生 `--accent-hover` 与 `--accent-glow`。存到 `SystemSetting` 的 `appearance.primaryColor`。

### 2.3 用色规则

- **主操作**（提交、新建、确认）只用 `--accent-primary`，一个页面同时最多 1 个主按钮。
- **状态色**（success/warning/danger）只用于状态徽章、提示框，**不用于装饰**。
- **危险操作**（删除、归档）按钮用 outline 风格 + danger 描边，避免全红实心按钮。
- **冲突严重度**`severity-*` 系列只在冲突/期限模块使用，不与其他色混用。

## 四、间距 & 圆角 & 阴影

| 项 | 值 | 用例 |
|---|---|---|
| 间距单位 | Tailwind 默认 4px 制 | `gap-2` = 8px，`gap-4` = 16px |
| 页面边距 | `px-8 py-6` | 主内容区四周留白 |
| 卡片内边距 | `p-5` | 20px |
| 圆角 - 小 | `rounded-md` (6px) | 输入框、徽章 |
| 圆角 - 中 | `rounded-lg` (8px) | 按钮、卡片 |
| 圆角 - 大 | `rounded-xl` (12px) | 模态、面板 |
| 阴影 - 卡片 | `shadow-[0_0_0_1px_var(--border-subtle)]` | 用描边代替阴影 |
| 阴影 - 悬浮 | `shadow-lg shadow-black/40` | 仅模态、下拉 |

**关键约束**：暗色主题下**少用阴影、多用描边**，否则看起来"脏"。

- **Topbar**：左侧 Logo + 当前模块面包屑 / 右侧搜索框（`⌘K`）+ 通知 + 头像菜单
- **Sidebar**：8 个一级菜单（仪表盘 / 收案 / 冲突 / 案件 / 客户 / 财务 / 日程 / 材料 / 设置），活动项左侧 2px 强调色亮条

### 6.2 响应式

| 断点 | 行为 |
|---|---|
| < 1024px | Sidebar 收起为图标 |
| < 768px | V1 暂不优化（律师主要桌面用），保证不崩 |

### 6.3 抽屉 vs 整页规则（决策 §10）

| 场景 | 形式 | 理由 |
|---|---|---|
| **新建** Intake / Matter / Client / Note / Task / FeeEntry | 右侧 `Sheet` 抽屉 | 不离开列表上下文，操作快，关闭无感 |
| **编辑**简单字段（改状态、改备注、加标签） | 内联编辑 / 行内弹层 | 一两个字段不值得开页面 |
| **编辑**复杂表单（案件信息、合同结算单） | 进入整页编辑 | 信息密度高，需要并行查看相关数据 |
| **查看**案件 / 客户 / 程序详情 | 整页（有 URL） | 可分享、可加书签、回流便 |
| **查看**只读小数据（某条 Note / Task 详情） | `Popover` 浮层 | 浮层即够 |
| **冲突检索**结果详情 | 整页 + 抽屉混合 | 检索结果整页，命中项详情可选择"在新抽屉看"或"跳目标案件" |

**抽屉规范**：
- 宽度：默认 `max-w-2xl`（约 672px）；多字段表单可 `max-w-3xl`
- 关闭：`Esc` / 点遮罩 / 右上 `×` / 提交成功后自动关闭
- 未保存离开：弹 confirm

| 刑事 | 一审/二审/再审/侦查/审查起诉/死刑复核/其他 | 被告人/被害人/自诉人/附带民事原告 | 隐藏 | 按代理程序联动：侦查→公安/监察；审查起诉→检察院；审判→法院；执行→监狱 |

**字段层级与视觉**：
- 必填项前红色 `*`；非必填无标记
- 字段标签右对齐，宽度固定 96px，输入区 flex
- 区段分隔线（"当事人"、"时间与人"等）字号 12px、灰色、左侧带 2px 短主色线
- 选中态按钮：背景 `--accent-primary` 30% 透明 + 1px `--accent-primary` 边 + 8px 主色 glow
- 未选按钮：透明背景 + `--border-default` 边；hover 时边变 `--accent-primary` 50%
- 主按钮"创建案件"：主色实心 + 12px glow，加载态变为带 spinner 的"创建中..."

**校验与体验**：
- 必填项空：字段红描边 + 字段下方 12px 红字提示，不弹 toast
- 客户/联系人 [+] 弹**二级抽屉**（叠在当前抽屉上、宽度 480px），新建后自动回填
- 案由搜索：300ms debounce，按 `category` + 模糊匹配 `name|shortName|pinyin|keywords`
- 关闭抽屉：若有改动弹 confirm "尚未保存的内容会丢失，确认关闭？"
- 创建成功：抽屉滑出，列表顶部插入新行并高亮 1.5 秒

| 交互 | 实现 |
|---|---|
| 全局快捷面板 `⌘K` | shadcn `Command` 组件，搜索案件/客户/跳页面 |
| 列表行悬停 | 背景色 `--bg-overlay`，左侧 2px 强调色亮条 |
| 数字滚动 | 仪表盘 KPI 数字加载时 400ms count-up |
| 状态切换 | 按钮微弹 + 状态徽章颜色 cross-fade |
| 抽屉打开 | 右侧 `Sheet` 滑入 200ms，背景 `bg-glass` 渐显 |
| 表单错误 | 字段红描边 + 左侧 shake 8px 一次 |
| Toast | `sonner` 右下角，4 秒后消失，可堆叠 |
| 骨架屏 | 列表/详情加载用 shadcn `Skeleton`，禁用 spinner |

**禁忌**：
- ❌ 不在表格行加 hover 动效（性能差且廉价）
- ❌ 不在按钮 hover 时改大小（用颜色 + 描边变化即可）
- ❌ 不用渐变色填充按钮背景
